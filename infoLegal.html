Información legal sobre el dominio público en libros

Cuando hablamos de dominio público "en libros" nos referimos a aquellas obras o creaciones literarias que pasados 70 años después de fallecido el autor, pueden ser usadas libremente por la sociedad sin ningún impedimento de copyright. Esta práctica en sí constituye un apoyo fundamental al crecimiento intelectual de toda la sociedad humana, ya que permite la utilización de estas creaciones libremente y por ello concede la oportunidad de que más personas que antes se veían limitadas económicamente (o por cualquier otra razón) accedan a los mismos.

Esto ocurre así en la mayoría de los países del mundo, aunque la cantidad de años requerida para que pase de un dominio al otro suele variar en cada uno de ellos. Generalmente, el plazo oscila entre los 50 y 70 años de producido el fallecimiento del autor. En países como Argentina, Turquía, Brasil, Israel, Nigeria, Rusia o muchos otros de la Unión Europea en sí misma, esto ocurre a los 70 años de producida la muerte del autor. En España, por ejemplo, esto ocurre a los 80 años.

En Argentina actualmente existe una base de datos de autores en la cual se pueden consultar las obras literarias en dominio público. Podemos acceder a ella en: http://www.dominiopublico.org.ar.

A pesar de ello, que un libro cumpla el plazo legal previsto en cada país no significa que pase a ser de dominio aboslutamente libre y público. Hay que investigar cada caso en particular para no incurrir en reproducciones que pueden resultar ilícitas. Existen casos, como por ejemplo "El Principito", en el cual si bien se ha cumplido el tiempo legal, los herederos de su autor han decidido convertir a cada personaje del libro y sus ilustraciones en marcas registradas, y por tanto protegidas. Esto implica que jamás pasarán al dominio público. Si bien la obra puede imprimirse o reproducirse sin pagarle nada a nadie, no se podrán usar sus personajes para hacer calendarios, tazas u otros productos de marketing. Tampoco se pueden utilizar los personajes para crear otras obras o las ilustraciones de forma independiente.

Licencia General Pública (GPL):

Mediante esta licencia podemos garantizar que los usuarios finales la libertad sobre el software y evitar que el mismo caiga en manos privadas evitando así su libre circulación, sin tener que pagar su utilización.

Tecnologías

Bootstrap:

Se ha elegido este framework como marco para generar el diseño y la interfaz del sitio web. Es una herramienta desarrollada por Twitter, actualmente es de licencia GPL. Este framework está compuesto por tres tecnologías de desarrollo, HTML, CSS & JavaScript.

HTML:

Se utiliza este lenguaje de Marcas de Hipertexto (HyperText Markup Language) para definir el significado y la estructura del contenido web, en combinación con las tecnologías de JavaScript (funcionalidad y comportamiento del sitio web) y Bootstrap (apariencia y presentación).

CSS:

“Hojas de Estilo en Cascada (del inglés Cascading Style Sheets) o CSS es el lenguaje de estilos utilizado para describir la presentación de documentos HTML o XML(incluyendo varios languages basados en XML como SVG, MathML o XHTML). CSS describe cómo debe ser renderizado el elemento estructurado en la pantalla, en papel, en el habla o en otros medios.”

Fuente: https://developer.mozilla.org/es/docs/Web/CSS

JavaScript:

“JavaScript (JS) es un lenguaje de programación ligero, interpretado, o compilado justo-atiempo (just-in-time) con funciones de primera clase. Si bien es más conocido como un lenguaje de scripting (secuencias de comandos) para páginas web, y es usado en muchos entornos fuera del navegador, tal como Node.js, Apache CouchDB y Adobe Acrobat, JavaScript es un lenguaje de programación basada en prototipos, multiparadigma, de un solo hilo, dinámico, con soporte para programación orientada a objetos, imperativa y eclarativa (por ejemplo programación funcional).”

Fuente: https://developer.mozilla.org/es/docs/Web/JavaScript

APIs de Conversión de Texto a Voz:

Mozilla ™ Web Speech API:

La “Web Speech API” de Mozilla nos permite incorporar datos de voz en aplicaciones web. Tiene dos partes: SpeechSynthesis (Texto a Voz) y SpeechRecognition (Reconocimiento de voz asíncrono). Esta API es compatible con la mayoría de los navegadores del mercado y puede ser implementada desde JavaScript, además de poseer decenas de idiomas.

Un ejemplo simple de cómo podría ser llamada la API sería:

function speak(text, language) {

var s = new SpeechSynthesisUtterance(text);

s.lang = language; speechSynthesis.speak(s);

}

“Mozilla” es una empresa pionera en el desarrollo de software libre y sin ánimos de lucros. Esta API en específico esta bajo la licencia Creative Commons Zero v1.0 Universal, que es un tipo de licencia para software libre, mediante la cual se puede copiar, modificar, distribuir e interpretar la obra, incluso para propósitos comerciales, sin pedir permiso.

Para más información : https://creativecommons.org/publicdomain/zero/1.0/deed.es

Repositorio en GitHub: https://github.com/mdn/web-speech-api
